# Terminology

## Software Agent
A software entity situated within a runtime environment that senses that environment and carries out its tasks, over time, in pursuit of its own goals and so as to effect what it senses in the future.

## Mobile Software Agent
A software agent that may be dispatched from a node in a network and transported to a remote node to carry out its tasks.

## Static Software Agent
A software agent that operates from a single node in a network, collaborating with other agents to devise solutions for complex tasks.

## Agent Migration
The process of transporting an agent from one node in a network to another.

## Agent Station
A runtime environment for static and mobile agents.

## Place
A space within an Agent Station where agents can operate according to assigned permissions.
